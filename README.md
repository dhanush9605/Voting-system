# Let's Vote

**Secure Democratic Voting**

A modern voting platform designed for transparent and secure elections.

## Features

- **Secure Voting**: End-to-end encrypted ballots with SHA-256 hash verification for complete transparency.
- **Identity Verification**: Photo-based registration ensures only eligible voters can participate.
- **Real-time Results**: Watch live tallies as votes are counted and verified by administrators.

## Technologies Used

This project is built with:

- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Node.js & npm installed

### Installation

1.  **Clone the repository**

    ```bash
    git clone <YOUR_GIT_URL>
    cd voting-system-main
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Start the development server**

    ```bash
    npm run dev
    ```

## Development

- **Edit Code**: The main application code is located in the `src` directory.
- **UI Components**: Reusable UI components are in `src/components/ui`.
- **Styling**: Global styles are defined in `src/index.css` and configured via `tailwind.config.ts`.
